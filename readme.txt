Setup / Compile
- set up the database student table
- open DatabaseOperations.java
- change the url, username, and password
- run the file

getAllStudents()
Accesses the database and runs through the student table, printing out all the information in each coloumn until the last row

addStudent()
Adds a new student into the students table, with input from the user

modifyStudentEmail(int, String)
In the students table, it will access the student with the given id and replace the email with the given parameter email

deleteUser(int)
In the students table, it will delete the student that has the matching id
